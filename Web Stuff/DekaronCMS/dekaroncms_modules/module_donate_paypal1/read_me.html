Description

This module will allow members to donate and get coins.

Step 1

Log in to your Website FTP server via your preferred File Transfer Protocol Client and upload the contents of the /upload_cms/ directory to the root of DekaronCMS directory.

Step 2 (Optional)

If you are using the API, copy the /upload_api/application/controllers/characterExistsPaypal.php & /upload_api/application/controllers/updateCashPaypal.php from the api folder to /your_api/application/controllers/ folder.

Step 3

Setup your API or SQL Server in the Settings -> Website -> Connection Type. Then API or SQL Server tab and fill in your details.

Step 4

Log in to the Admin panel and visit the Modules. Click the Install button and follow the steps.

Step 5

Follow the steps in the "paypal_developer.html". (Included in the zip)

Step 6

Go to Donate Paypal Settings and fill in your details and setup anything you like. Please note that the Donate Paypal Items are pre-populated with some items, feel free to change them according to your server settings. By default the paypal mode is set as "Sandbox (for Testing)"!

Step 7

Go to Menu -> Create Link
-> Title: Donate
-> URL: donate_paypal
-> Menu location: Top and/or Side
-> Visibility mode: Visible to everyone
-> Internal direct link: No
Save Menu

Step 8

Go to yourdomain.com/donate or clicking the Donate button on your website and test your donate module.
NOTE: If you dont see the donate link, refresh your page.
