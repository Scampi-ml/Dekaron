Description

This module will allow new users to register on your website using the API or Local connection.

Step 1

Log in to your Website FTP server via your preferred File Transfer Protocol Client and upload the contents of the /upload_cms/ directory to the root of DekaronCMS directory.

Step 2 (Optional)

If you are using the API, copy the /upload_api/application/controllers/register.php from the api folder to /your_api/application/controllers/ folder.

Step 3 (Recommended)

Please check /upload_cms/application/modules/register/models/register_model.php on line 66 for the correct data matching your database. You may need to do this also for the API file in /upload_api/application/controllers/register.php on line 33.

Step 4

Setup your API or SQL Server in the Settings -> Website -> Connection Type. Then API or SQL Server tab and fill in your details.

Step 5

Log in to the Admin panel and visit the Modules. Click the Install button and follow the steps.

Step 6

Go to Menu -> Create Link
-> Title: Register
-> URL: register
-> Menu location: Top and/or Side
-> Visibility mode: Visible to everyone
-> Internal direct link: No
Save Menu

Step 7

Go to yourdomain.com/register or clicking the Register button on your website and test your register module.
NOTE: If you dont see the register link, refresh your page.
